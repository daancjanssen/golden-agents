# Golden Agents Adapter

## requirements

```
$ pip3 install requests
$ pip3 install flask
$ pip3 install flask_cors
```

## running the adapter

```
# make sure it is executable
# chmod u+x ./runs.sh
$ ./run.sh
```

## frontend, .env.development
```
REACT_APP_DATASETS_API="http://127.0.0.1:5000/ga"
```
